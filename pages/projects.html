Projects

During my studies, I have worked on various projects that allowed me to apply my programming skills and gain hands-on experience in software and hardware development. Below are highlights of two major projects I contributed to during my first year of study.

Flight Game

A collaborative group project and my first programming endeavor, developed as part of a school assignment. The game uses a database containing global airport information and was built with MySQL, JavaScript, Python, HTML, and CSS. Players navigate between airports while managing resources and weather conditions.

I primarily worked on the backend, connecting the JavaScript game loop with Python-based APIs. My tasks included implementing the game loop in JavaScript, designing APIs to communicate with a MySQL database and OpenWeatherMap for real-time weather data, and integrating these functionalities into the game.

Heart Rate Monitoring Device

A health-focused project that measures stress levels through heart rate monitoring. Designed for at-home or office use, this device provides real-time feedback using Raspberry Pi Pico W, Raspberry Pi, and Kubios. The system captures PPG signals with a Crowtail Pulse Sensor and calculates heart rate and heart rate variability.

I contributed by developing the class structure and main loop, implementing menus, designing the peak detection algorithm, calculating heart rate and HRV, and establishing communication with Kubios for advanced HRV analysis. Results were displayed on an OLED screen, with local data saved and retrievable for review.
